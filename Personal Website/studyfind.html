StudyFind
Junior Design
This project was a web scraper that collected and stored information on currently running clinical trials. This information was gathered from clinicaltrials.gov, a government website on which all clinical trials must register. Our system can be run periodically to check for new studies and add them to the cloud-based database that the scraper is linked to. Additionally, the scraper generates two new fields of information using Natural Language Processing. These fields are keywords and a one sentence summary, both generated from the study's description.
